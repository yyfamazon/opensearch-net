    Class CombinedFieldsQueryDescriptor<T>

    The combined_fields query supports searching multiple text fields as if their contents had been indexed into one combined field. The query analyzes the provided text before performing a search, and it scores documents using a term-centric approach that treats the matched fields as a single field.

    All queried fields must have the same search analyzer. Only text fields are supported.
